Présentation

Fullstack est une application exemple qui permet de mettre en exergue les différents attendus d’une application au sein de l’Urssaf Caisse Nationale.

Cette application contient un ensemble d’exemples permettant au développeur d’avoir une base de développement fiable dans l’écosystème de la DSI. Elle contient différents composants (SPA-Intranet et backend). L'IHM intranet SPA basée sur Angular et Bootstrap propose notamment :
  • Le respect des chartes graphique et ergonomique
  • Une implémentation de services REST OpenAPI
  • Une implémentation de la sécurisation Prisme
  • La communication inter IHM avec Com-SPA
  • Des exemples de tableaux Aggrid

Fullstack se déploie sur les 3 plateformes principales : HAWAI, SX et PFS. Vous y trouverez donc des exemples de jenkinsFile pour chacune de ces plateformes.

Cette application est maintenue par l’équipe SDAT > ACT (anciennement BT-AEA), qui l’utilise au quotidien pour s’assurer que les évolutions du SI soient compatibles et cohérentes avec le reste de l’écosystème.

Le code source est ouvert. N’hésitez pas à nous faire part d’anomalies ou nous partager d’éventuelles évolutions. Vous trouverez un ensemble de liens vers les différents Git, ReadtheDoc, ou notre mail de contact sur les différents onglets de cette page d’accueil.
